## Install

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```bash
npm install
```

## Run

- **Option 1 - Docker (recommended)**

    - To launch the full stack, use the [`docker-compose.yml`](../docker-compose.yml) in the root folder.

        ```bash
        cd ..
        docker compose up
        ```

- **Option 2 - Manual**

    - Start the development server:

        ```bash
        npm run dev
        ```

## Test

```bash
npm run test
```
